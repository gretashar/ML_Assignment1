The intended purpose of this README is to describe each file for the Assignment#1: Supervised Learning and to provide instruction to run the code.

*********************************************************************************************************************************
Data Used:

Heart Analysis & Prediction Dataset
https://www.kaggle.com/datasets/rashikrahmanpritom/heart-attack-analysis-prediction-dataset

Stroke Prediction Dataset
https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset

*********************************************************************************************************************************

Code:


-----------------------------------------------------------------------------------------------------
1. Boosting.py - Will run boosting algorithm.

	python3 Boosting.py 

2. NN.py - Will run Neural Networks algorithm.

	python3 NN.py 

3. DT.py - Will run Decision Tree algorithm.

	python3 DT.py 

4. KNN.py - Will run KNN algorithm.

	python3 KNN.py 

5. SVM.py - Will run SVM algorithm.

	python3 SVM.py 

-----------------------------------------------------------------------------------------------------
6. Balance.py - over/undersamples unbalanced dataset

7. Plot.py - provides several plotting functions

8. learning_curve.py - functions for learning and validation curves
